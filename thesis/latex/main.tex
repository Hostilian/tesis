\chapter*{Abstract}
This thesis investigates the implementation of space-based economic intelligence systems using civilian satellite APIs. By constructing an automated pipeline querying GEE and CDSE collections, spectral anomalies are extracted and analyzed via unsupervised Isolation Forests and temporal Z-scores.
